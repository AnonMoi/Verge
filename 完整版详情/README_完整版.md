# Verge — 时间循环塔防游戏

> **「钟摆未眠，黎明将至」**
>
> 一款以"昼夜时间循环"为核心机制的创新 2D 塔防游戏。
> Godot 4.x 开发 | Demo v1.0 已可运行

---

## 🎮 项目简介

**Verge** 融合了植物大战僵尸式的路径塔防玩法和独特的**时间循环系统**。玩家在白天部署角色、发展经济；黄昏进行最终备战；夜晚抵御怪潮；黎明清屏所有敌人——然后一切重新开始，迎接更艰难的下一天。

### 核心特色

| 特色 | 说明 |
|------|------|
| ⏰ **时间循环** | 白天(90s) → 黄昏(30s) → 夜晚(90s) → 黎明(清屏) → 下一天 |
| 👥 **角色塔防** | 3 种角色：先锋(近战快攻) / 重装(肉盾) / 狙击(远程高台) |
| 🌅 **黎明清屏** | 黎明时刻全屏白闪，所有敌人消散，角色回血，极致爽感 |
| 💰 **经济策略** | 金矿产出 + 击杀奖励，夜晚金矿停产迫使主动作战 |
| 🎨 **像素美术** | AnimatedSprite2D 角色动画，暗色系 UI 风格 |

## 🚀 快速开始

### 环境要求
- **Godot Engine 4.x**（推荐 4.7）
- Windows / macOS / Linux

### 运行步骤
1. 安装 [Godot Engine](https://godotengine.org/download/)
2. 打开 `project.godot`
3. 按 `F5` 运行游戏

### 操作说明
| 操作 | 说明 |
|------|------|
| 点击底部角色按钮 | 选中角色，再点击地图部署 |
| `Esc` | 暂停菜单 |
| 倍速按钮 | 1x ↔ 2x 切换 |
| `F3` (测试) | 直接触发胜利结算 |
| `F4` (测试) | 直接触发失败结算 |

## 🏗️ 技术架构

### Autoload 单例（6个）
| 单例 | 职责 |
|------|------|
| `Constants` | 全局常量（时间/网格/类型） |
| `SignalBus` | 信号总线（7类24个信号） |
| `GameManager` | 游戏状态 + 关卡统计 |
| `TimeCycle` | 时间循环 + 阶段切换 |
| `Economy` | 金币 + 金矿 + 击杀奖励 |
| `Resolution` | 分辨率适配（1280×720） |

### CanvasLayer 层级
| 层级 | 用途 |
|------|------|
| 50 | 底部部署栏 |
| 100 | 黎明特效 |
| 150 | 暂停菜单 |
| 180 | 胜利结算 |
| 200 | 失败结算 |

## 🔮 完整版规划（见策划书）

完整版策划书详细描述了以下系统的设计：

- ⏰ **时间循环系统** — 白天/黄昏/夜晚/黎明 四阶段循环

- 🔥 **元素反应系统** — 7元素（火/冰/雷/水/风/岩/草）联动反应
- 🃏 **卡牌系统** — 40+张卡牌，含融合技/符文槽/变异机制
- 🎲 **Roguelite 肉鸽** — 祝福/诅咒/随机事件/动态价格
- 👹 **敌人图鉴** — 25+种敌人（含Boss/元素怪/特殊机制）
- 🗺️ **地形与天气** — 6种地形 × 7种天气的动态环境
- 💰 **多元经济** — 金币/EP/声望 三条经济线
